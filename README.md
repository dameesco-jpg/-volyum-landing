# VOLYUM Landing Page

A striking landing page for VOLYUM featuring a disco ball video background with the VOLYUM logo and tagline.

## 🚀 Deploy to GitHub Pages

### Step 1: Create a New Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right and select "New repository"
3. Name it: `volyum-landing` (or any name you prefer)
4. Make it **Public**
5. ✅ Check "Add a README file"
6. Click "Create repository"

### Step 2: Upload Your Files
1. In your new repository, click "Add file" → "Upload files"
2. Drag and drop these 3 files:
   - `index.html` (the landing page)
   - `volyum-logo.png` (the logo image)
   - `disco-ball.mp4` (the video background - 11MB)
3. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to your repository Settings
2. Scroll down to "Pages" in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait 1-2 minutes for deployment

### Step 4: View Your Live Site
Your site will be live at:
```
https://YOUR-USERNAME.github.io/volyum-landing/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

## 📁 Files Included

- `index.html` - Main landing page with animations and effects
- `volyum-logo.png` - VOLYUM logo (white on transparent)
- `disco-ball.mp4` - Disco ball video background (compressed, 11MB)

## 🎨 Features

- Full-screen looping disco ball video background
- Animated VOLYUM logo with glowing effects
- Gradient text tagline: "FUELED BY ARTISTS. POWERED BY BRANDS."
- Responsive design (works on mobile and desktop)
- Smooth parallax mouse movement
- Hover effects and animations

## 🔧 Customization

To edit the page, simply modify `index.html`:
- Change colors in the CSS `:root` variables
- Adjust animation speeds in the `@keyframes` sections
- Modify text in the `.tagline` section

## 📝 Notes

- The video file is optimized for web (11MB) for fast loading
- All files must be in the same directory for the page to work properly
- Works best on modern browsers with video support
